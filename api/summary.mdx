<Note>
Endpoint: `GET /api/summary`

**Authentication Required**
</Note>

Retrieve a monthly activity summary for the currently authenticated user. This endpoint generates a focused snapshot of GitHub activity for a specific month and year.

## Authentication

<Warning>
This endpoint requires authentication. Users must be logged in via GitHub OAuth.
</Warning>

The endpoint:
1. Validates the user has an active session
2. Retrieves the GitHub access token from the session
3. Fetches the authenticated user's GitHub profile
4. Generates a monthly wrapped summary

## Request

### Query Parameters

<ParamField query="year" type="number" default="Current year">
  Year to generate summary for (2008 - current year)
  
  Example: `2024`
</ParamField>

<ParamField query="month" type="number" default="Current month">
  Month to generate summary for (1-12)
  
  Example: `3` (March)
</ParamField>

<ParamField query="accessToken" type="string">
  Optional GitHub personal access token override
  
  **Note**: Typically not needed as the session token is used automatically.
</ParamField>

### Validation Rules

- `year`: Integer between 2008 and current year
- `month`: Integer between 1 and 12

## Response

<ResponseField name="data" type="UserWrappedData">
  User wrapped data scoped to the specified month
  
  The structure is similar to `/api/wrapped/user` but focused on a single month's activity.
  
  See [User Wrapped](/api/user-wrapped#response) for the full data structure.
</ResponseField>

## Examples

### Specific Month Request

<CodeGroup>
```bash cURL
curl "https://your-domain.com/api/summary?year=2024&month=3" \
  -H "Cookie: better-auth.session_token=..."
```

```javascript JavaScript
// Get March 2024 summary
const response = await fetch(
  'https://your-domain.com/api/summary?year=2024&month=3',
  { credentials: 'include' }
);

const { data } = await response.json();
console.log(`Commits in March: ${data.overview.totalCommits}`);
console.log(`PRs in March: ${data.overview.totalPRs}`);
```

```typescript TypeScript with React
import { useState, useEffect } from 'react';
import { authClient } from '@/lib/auth-client';

function MonthlySummary() {
  const [summary, setSummary] = useState(null);
  const { data: session } = authClient.useSession();
  const [year, setYear] = useState(2024);
  const [month, setMonth] = useState(3);
  
  useEffect(() => {
    if (!session) return;
    
    fetch(`/api/summary?year=${year}&month=${month}`)
      .then(r => r.json())
      .then(result => setSummary(result.data));
  }, [session, year, month]);
  
  if (!session) return <div>Please log in</div>;
  if (!summary) return <div>Loading...</div>;
  
  return (
    <div>
      <h2>Summary for {year}-{month.toString().padStart(2, '0')}</h2>
      <p>Commits: {summary.overview.totalCommits}</p>
      <p>PRs: {summary.overview.totalPRs}</p>
      <p>Issues: {summary.overview.totalIssues}</p>
    </div>
  );
}
```
</CodeGroup>

## Use Cases

### Monthly Activity Dashboard

```typescript
function MonthlyActivity() {
  const [summaries, setSummaries] = useState([]);
  const currentYear = 2024;
  
  useEffect(() => {
    // Fetch all 12 months
    Promise.all(
      Array.from({ length: 12 }, (_, i) => i + 1).map(month =>
        fetch(`/api/summary?year=${currentYear}&month=${month}`)
          .then(r => r.json())
      )
    ).then(results => {
      setSummaries(results.map(r => r.data));
    });
  }, []);
  
  return (
    <div>
      <h1>{currentYear} Monthly Activity</h1>
      {summaries.map((summary, i) => (
        <MonthCard 
          key={i}
          month={i + 1}
          commits={summary.overview.totalCommits}
          prs={summary.overview.totalPRs}
        />
      ))}
    </div>
  );
}
```

### Current vs Previous Month Comparison

```javascript
async function compareMonths() {
  const now = new Date();
  const currentMonth = now.getMonth() + 1;
  const currentYear = now.getFullYear();
  
  const previousMonth = currentMonth === 1 ? 12 : currentMonth - 1;
  const previousYear = currentMonth === 1 ? currentYear - 1 : currentYear;
  
  const [current, previous] = await Promise.all([
    fetch(`/api/summary?year=${currentYear}&month=${currentMonth}`)
      .then(r => r.json()),
    fetch(`/api/summary?year=${previousYear}&month=${previousMonth}`)
      .then(r => r.json())
  ]);
  
  const commitChange = 
    current.data.overview.totalCommits - previous.data.overview.totalCommits;
  
  console.log(`Commit change: ${commitChange > 0 ? '+' : ''}${commitChange}`);
}
```

### Activity Calendar

```typescript
function ActivityCalendar() {
  const [monthData, setMonthData] = useState(null);
  const [selectedMonth, setSelectedMonth] = useState(new Date().getMonth() + 1);
  const [selectedYear, setSelectedYear] = useState(new Date().getFullYear());
  
  useEffect(() => {
    fetch(`/api/summary?year=${selectedYear}&month=${selectedMonth}`)
      .then(r => r.json())
      .then(result => setMonthData(result.data));
  }, [selectedMonth, selectedYear]);
  
  return (
    <div>
      <MonthPicker 
        month={selectedMonth}
        year={selectedYear}
        onChange={(m, y) => {
          setSelectedMonth(m);
          setSelectedYear(y);
        }}
      />
      
      {monthData && (
        <>
          <ActivityHeatmap data={monthData.dailyActivity} />
          <TopRepos repos={monthData.topRepos} />
          <LanguageBreakdown languages={monthData.overview.topLanguages} />
        </>
      )}
    </div>
  );
}
```

## Implementation Details

The endpoint uses the `generateUserWrappedForMonth` analytics method:

```typescript
const analytics = new AnalyticsEngine(github);
const wrappedData = await analytics.generateUserWrappedForMonth(
  ghUser.login,
  year,
  month
);
```

This method filters all user activity to the specified month, providing focused insights.

## Caching

This endpoint does **not cache** responses because:
- Data is user-specific and private
- Monthly summaries are typically accessed for current/recent months
- Users expect real-time data for their own accounts
